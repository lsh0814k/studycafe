# 스터디 모집
### 프로젝트 구성
    - java 17
    - Spring boot 3.2.0
    - Spring Data Jpa
    - Spring Security6
    - H2
    - thymeleaf
    - Querydsl
### 기능 요약
    1. 회원 가입
    2. 스터디 등록, 수정, 삭제
    3. 모임 등록, 수정, 삭제
    4. 알림
        4-1 관심 있는 지역, 주제에 맞는 스터디 또는 모임이 등록된 경우 알림이 생성된다.
